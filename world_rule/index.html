PUBLIC GENERATION SPEC

World Generation Algorithm

NiceChunk terrain is generated from a public seed pipeline. The same seed and coordinates produce the same terrain profile, biome, water, caves, trees, and block query results.

Deterministic Inspectable Runtime rules
01

Seeded world state

A text seed is hashed once, then mixed into every coordinate sample.

02

Heightfield first

Continents, oceans, relief, rivers, dunes, and mountains produce the base terrain.

03

Climate and biome pass

Temperature, humidity, slope, altitude, water, and special masks select the biome.

04

Block query rules

The engine answers block requests from depth, fluid, cave, surface, and vegetation rules.
